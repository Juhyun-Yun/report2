# 과학탐구 활동 보고서 — 공유 설치 안내

이 앱은 **선생님마다 자기 시트를 각자 연결**하는 구조로 만들었습니다.
학생 명단과 제출물(PDF·그래프)은 전부 **그 선생님의 구글 계정에만** 저장되어, 선생님끼리 데이터가 섞이지 않습니다. 다른 선생님은 앱 코드(`index.html`)를 만질 필요가 없습니다.

---

## A. 처음 만드는 사람(관리자)이 한 번 하는 일

### 1) 앱 올리기 (코드 공유 없이 링크만 나눠주기)
- `index.html`을 정적 호스팅에 올립니다. 가장 쉬운 방법은 **GitHub Pages**입니다.
  - 새 저장소 → `index.html` 업로드 → Settings ▸ Pages ▸ Branch `main` 선택 → 저장
  - 몇 분 뒤 `https://아이디.github.io/저장소이름/` 주소가 생깁니다. → 이 주소를 선생님들에게 나눠줍니다.
- (그 외 Netlify, Vercel, 학교 홈페이지 등 어디든 `index.html` 하나만 올리면 됩니다.)

### 2) 선생님용 "템플릿 시트" 만들기
1. 빈 구글 스프레드시트를 하나 만듭니다. (예: `과학탐구 보고서 - 템플릿`)
2. 메뉴 **확장 프로그램 ▸ Apps Script** → 기본 `Code.gs` 내용을 모두 지우고, 함께 드린 **`Code.gs`** 내용을 붙여넣고 저장합니다.
3. 시트로 돌아와 새로고침하면 상단에 **📋 보고서 앱** 메뉴가 생깁니다.
   → **📋 보고서 앱 ▸ 초기 설정(탭 만들기)** 클릭 → `명단`·`제출` 탭이 자동 생성됩니다.
4. (선택) 시트에 **`📖 선생님 가이드`** 탭을 추가해 아래 B의 4단계를 그림과 함께 적어두면 더 친절합니다.

### 3) "사본 만들기" 링크 만들어 두기
- 템플릿 시트 주소가 `https://docs.google.com/spreadsheets/d/시트ID/edit` 라면,
  끝부분 `/edit...`을 **`/copy`** 로 바꾼 주소
  `https://docs.google.com/spreadsheets/d/시트ID/copy`
  가 "사본 만들기" 링크입니다. (선생님이 누르면 바로 자기 드라이브로 복사됨)
- 이 링크를 `index.html` 상단의 `TEMPLATE_COPY_URL = ''` 안에 붙여넣으면,
  앱 설정 화면에 **`① 템플릿 시트 사본 만들기`** 버튼이 자동으로 나타납니다.
  (비워두면 버튼만 숨겨질 뿐, 동작에는 문제없습니다.)

---

## B. 사용하는 선생님이 한 번 하는 일 (4단계, 약 3분)

1. **템플릿 시트 사본 만들기** — 받은 "사본 만들기" 링크를 누르고 *사본 만들기*.
2. **명단 입력** — `명단` 탭에 번호·이름을 적습니다. (팀/모둠으로 운영하면 `팀` 칸도 채우기)
3. **웹 앱 한 번 배포**
   - 메뉴 **확장 프로그램 ▸ Apps Script ▸ 배포 ▸ 새 배포**
   - 유형 **웹 앱** → 실행 **나** → 액세스 **모든 사용자** → **배포**
   - 처음이면 권한 화면에서 **고급 ▸ (안전하지 않음)으로 이동 ▸ 허용**
     *(내가 만든 내 시트라서 안전합니다.)*
4. **주소 붙여넣기** — 배포 후 나오는 **웹 앱 URL**(끝이 `/exec`)을 복사해
   앱의 **[선생님 설정]** 칸에 붙여넣고 **저장**.

> 끝났습니다. 이제 학생은 드롭다운에서 이름을 골라 시작하고,
> 작성한 보고서를 **[제출]** 하면 선생님 시트의 `제출` 탭과 드라이브 폴더에 자동으로 쌓입니다.

### 교실 기기(태블릿)에서 더 간단하게
- 앱 주소 뒤에 `?api=웹앱주소` 를 붙인 링크를 만들어
  교실 기기에서 한 번만 열어두면, 설정 화면 없이 바로 **이름 선택**으로 들어갑니다.
  예) `https://아이디.github.io/저장소/?api=https://script.google.com/macros/s/XXXX/exec`
  (한 번 열면 그 기기에 저장되어, 다음부터는 그냥 앱 주소만 열어도 됩니다.)

## 자주 묻는 것
- **데이터는 누가 보나요?** 각 선생님 본인만 봅니다. 시트도 PDF도 그 선생님 드라이브에 저장됩니다.
- **명단 열 순서를 바꿔도 되나요?** 됩니다. `번호/이름/팀` 글자가 들어간 열 제목을 자동으로 찾습니다.
- **그래프 이미지가 안 보여요.** 잠시 후 새로고침하면 표시됩니다. 안 보여도 `보고서(PDF)` 링크 안에 전체 내용이 들어 있습니다.
- **코드(Code.gs)를 고쳤어요.** 배포 ▸ 배포 관리 ▸ 편집(연필) ▸ 버전 **새 버전** ▸ 배포 를 해야 반영됩니다.
